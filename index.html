Pseudo Classes

Sep 6th, 2013

I am still not able to solve my rails error so I am spending more time on learning about CSS. This is probably very basic knowledge, something frontend developers do in their sleep, but I feel pretty “Wheee” now that I am able to target certain list items.

:nth-child will target child-elements based on their position.

For example I have an unordered list and want to add a styling only to the third list-element. All I have to do is pass the number 3 into the parenthesis :

li:nth-child(3) {
    color: white;
    background-color: black;
}

I can even do more with using expressions such as: (an+b).

b will determine the start-position and a will then select every “put a number here” element on top of that. We don’t have to worry about n, it will just stay the way it is. Example: I have an unordered list and want to apply a styling to the third list item AND every 2nd item from that point on. I would use the (an+b) formula and just pass in my numbers 2 (for a) and 3 (for b).

li:nth-child(2n+3) {
    color: white;
    background-color: black;
}

NoMethodHorror

Aug 30th, 2013

In my app I want to accept a pending friend request. But when I hit the “accept” button I get a NoMethodError in UserFriendship#edit: undefined method `requested?‘ for nil:NilClass.

views => user_friendship => edit.html.erb:

<% if @user_friendship.requested? %>
  Do you really want to be friends with <%= @friend.first_name %>?
<% end %>

<% if @user_friendship.accepted? %>
  You are now friends with <%= @friend.first_name %>s
<% end %>

I still haven’t figured out why @user_friendship returns nil. But I read a bit about NoMethodErrors and I found this to be a cute descriptive example:

So, you press the button on the forehead of the first person and the person says, “Julian McNeil.” Ok good. Then you press the button on the forehead of the second person and the person says, “Sally Reynolds.” Good again. When you get to the 3rd person, there is no person. Just a button on the floor that says full name. You press the button anyways. Nothing. How can that person say their full name if there is no person? That 3rd person is nil. -Source

Then I ended up reading a bit about the method try(). If I do @user_friendship.try(:requested?) I can “escape” that NoMethodError. It´s not going to solve my problem but at least I got my tests to pass for a moment ;).
